{% load i18n %} {% comment %} Reusable iOS-style roller (wheel) date-time picker. Auto-enhances every on the page: the native input is hidden and replaced by a tappable display field. Tapping it opens a bottom sheet with wheels for month / day / year / hour / minute. Values are written back in the datetime-local format (YYYY-MM-DDTHH:MM) so normal form submission is unchanged. The wheels are driven entirely by JS (transform-based), not native scrolling, so dragging can never chain to the page underneath and the sheet only dismisses via the Cancel / Done buttons. {% endcomment %}
