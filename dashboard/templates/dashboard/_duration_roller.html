{% load i18n %} {% comment %} Reusable iOS-style roller (wheel) duration picker. Auto-enhances every on the page (a hidden input holding a duration in whole minutes). The native input is hidden and replaced by a tappable display field. Tapping it opens a bottom sheet with wheels for hours and minutes. The value is written back as total minutes so normal form submission is unchanged. The wheels are driven entirely by JS (transform-based), not native scrolling, so dragging can never chain to the page underneath and the sheet only dismisses via the Cancel / Done buttons. Mirrors _time_roller.html. {% endcomment %}
